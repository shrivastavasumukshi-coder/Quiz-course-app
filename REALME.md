# Quiz Application

This is a full-stack quiz application built as part of a frontend and backend test.

## Features
- Multiple-choice quiz with 5 questions
- One question displayed at a time
- Progress bar (current question / total)
- Immediate correct/incorrect feedback
- Previous and Next navigation
- Progress saved using localStorage
- Quiz resume on refresh
- Final score summary

## Frontend
- HTML
- CSS
- JavaScript

## Backend
- Node.js
- Express.js
- In-memory course and progress storage
- API key based authentication
- Input validation and error handling
